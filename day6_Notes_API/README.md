# Day 6 - FastAPI CRUD Notes API

## Concepts Learned

* FastAPI
* SQLAlchemy ORM
* PostgreSQL Integration
* CRUD Operations
* REST APIs
* Swagger Documentation

## Endpoints

### GET /

Health check endpoint

### POST /notes

Create a new note

### GET /notes

Retrieve all notes

### GET /notes/{id}

Retrieve a specific note

### DELETE /notes/{id}

Delete a note

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* Docker

## Project Summary

Built a Notes Management API that stores and retrieves notes from a PostgreSQL database running inside Docker. Implemented CRUD operations using FastAPI and SQLAlchemy.
